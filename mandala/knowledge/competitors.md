# Kompetitor & Landscape

## Direct Competitors (AI Chat Agent Indonesia)

### Kata.ai
- Enterprise-focused, harga tinggi
- Lebih ke chatbot NLP daripada AI conversational
- Kelebihan mereka: brand besar, established
- Kelemahan: mahal, setup lama, kurang personal

### Qontak (Mekari)
- CRM + WhatsApp API
- Lebih ke pipeline management daripada AI conversation
- Kelebihan: all-in-one CRM
- Kelemahan: bukan AI-first, chatbot-nya basic

### SleekFlow
- WhatsApp Business API + basic automation
- Kelebihan: multi-channel, clean UI
- Kelemahan: AI-nya basic, pricing USD (mahal buat UMKM)

## Indirect Competitors

### Hire Admin Manual
- Cost: Rp 3-5jt/bulan per orang
- Pro: human touch, flexible
- Con: gak 24/7, inconsistent, data hilang kalau resign

### Template Chatbot (ManyChat, Chatfuel)
- Cost: free - $15/bulan
- Pro: murah, easy setup
- Con: template/keyword-based, terasa robotic, gak bisa konteks

### VA (Virtual Assistant)
- Cost: Rp 2-4jt/bulan
- Pro: human, bisa multitask
- Con: gak 24/7, butuh training, turnover tinggi

## Positioning Jadisatu

Jadisatu = sweet spot antara:
- Kualitas enterprise AI (kayak Kata.ai)
- Harga UMKM-friendly (di bawah cost admin)
- Setup terima beres (gak perlu teknis)
- Natural conversation (gak terasa bot)
