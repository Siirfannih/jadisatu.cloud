# Skill: Reporting

Generate status report dan analytics. Tersedia di CEO Mode.

## Morning Briefing (7am WITA)

Format:
```
Selamat pagi 👋

Status hari ini:
- Server: [ok/warning]
- Chat kemarin: [X] conversations, [Y] new leads
- Hot leads: [list nama + score]
- Task pending: [jumlah]
- Perlu perhatian: [hal kritis]
```

## On-Demand Status Report

Trigger: Owner bilang "status" atau "progress"
Format:
```
Status [tanggal jam]
━━━━━━━━━━━━━━━━
Server: [detail]
Active chats: [jumlah] | Mandala handling: [jumlah]
Leads pipeline: [hot/warm/cold counts]
Tasks: [active/done/pending]
Revenue MTD: [jika available]
```

## Conversation Analytics

- Total conversations per hari/minggu
- Mandala vs Owner handling ratio
- Average response time
- Conversion rate (lead → demo → closing)
- Top objections encountered
- Score distribution of leads

## Weekly Summary (Setiap Senin)

Ringkasan minggu lalu:
- Leads masuk: [jumlah]
- Qualified (score 5+): [jumlah]
- Closing: [jumlah]
- Revenue: [total]
- Top performing conversation patterns
- Improvement areas
