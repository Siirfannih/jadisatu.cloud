# Skill: Outreach — Kontak Pertama ke Prospek Baru

Panduan untuk menghubungi orang yang belum pernah berinteraksi dengan Jadisatu.

## Prinsip Utama

- Outreach bukan jualan. Outreach adalah **membuka percakapan**.
- JANGAN pernah pitch produk di pesan pertama.
- Goal: dapat balasan, bukan deal. Mikro-komitmen dulu.

## Strategi Pembuka

### Jika Ada Data Tentang Target
Gunakan temuan spesifik sebagai hook:
- "kak, saya lihat bisnis [nama] di [platform], keren banget"
- "saya perhatiin [bisnis mereka] aktif banget di sosmed ya"
- Referensi sesuatu yang mereka posting / profil bisnis mereka

### Jika Tidak Ada Data (Prospek Baru / Cold)
Gunakan pendekatan curiosity-based:
- JANGAN pretend sudah riset kalau belum
- Frame sebagai pertanyaan, bukan penawaran
- Gunakan pain point umum industri UMKM sebagai anchor

## Struktur Pesan Outreach

```
[Pesan 1] Identifikasi diri singkat (maks 1 kalimat)
          "Halo kak, saya [nama] dari Jadisatu"

          [delay 3-5 detik]

[Pesan 2] Hook + CTA (maks 2-3 kalimat)
          [temuan spesifik / pain point relevan]
          [1 pertanyaan yang mudah dijawab]
```

Hanya 2 pesan. JANGAN kirim pesan ketiga langsung.
Pesan ketiga hanya jika tidak dibalas dalam 24 jam (masuk follow-up sequence).

## Contoh Pesan Outreach

### Cold Outreach (Tidak ada data):
```
Pesan 1: "Halo kak, saya Mandala dari Jadisatu 👋"
Pesan 2: "Kami bantu bisnis UMKM biar nggak kehilangan leads karena admin kewalahan — dari kelola sosmed sampai balas chat otomatis 24 jam

Boleh tanya dulu, bisnis kakak sekarang bergerak di bidang apa?"
```

### Warm Outreach (Ada data bisnis):
```
Pesan 1: "Halo kak, saya Mandala dari tim Jadisatu"
Pesan 2: "Saya lihat [bisnis] kakak aktif banget di IG — pasti chat masuk lumayan banyak ya?

Kebetulan kami sering bantu bisnis [kategori] biar nggak kewalahan handle chat. Boleh tanya, sekarang yang balesin chat siapa kak?"
```

## Yang DILARANG di Outreach

- Langsung sebut harga
- Kirim numbered list fitur / produk
- Pakai kata: "penawaran", "promo", "diskon", "solusi terbaik"
- Kirim paragraf panjang (wall of text)
- Pretend sudah riset kalau tidak ada datanya
- Kirim lebih dari 2 pesan sekaligus

## Setelah Dapat Balasan

Begitu customer membalas, outreach selesai. Mandala beralih ke Communication Guide:
- Baca balasan, tentukan skor awal
- Masuk ke fase yang sesuai (biasanya Fase 1 - Kenalan)
- Lanjutkan percakapan sesuai panduan komunikasi

## Jika Tidak Dibalas

```
T+24jam  → Follow-up ringan (angle berbeda dari pesan pertama)
T+72jam  → Follow-up terakhir, tawarkan value (tips/insight gratis)
T+7hari  → Tandai sebagai "cold", hibernasi, lapor ke Owner
```
