Integrations — Canva

Untuk Carousel Generator: sambungkan akun Canva (OAuth) dan/atau isi link template. Warna, font, dan ukuran kanvas diatur di Carousel Generator → Manual Canvas (Buka Configurator).

Connect to Canva (OAuth)

Sambungkan akun Canva agar sistem bisa membuat & mengedit desain carousel lewat Canva. Tanpa ini, Publish to Canva hanya membuka link.

Memeriksa…

Akun Canva yang dipakai saat connect boleh berbeda dengan akun yang mendaftar integrasi. Jika Canva menampilkan "ID klien tidak valid", perbarui CANVA_CLIENT_ID di Supabase → Edge Functions → Secrets agar sama dengan Client ID dari Canva Developer Portal.

Jika Connect gagal di Chrome, coba browser lain (mis. Brave).

Via Canva Design (link template)

Paste link desain Canva. Sistem akan mempelajari style & layout desain ini untuk membuat carousel baru dengan tampilan serupa. Link ini juga dipakai untuk "Open in Canva" jika perlu.

Belum diset

Brand Config (warna, font, ukuran kanvas) tidak di sini. Atur di halaman Carousel Generator → kartu Manual Canvas → Buka Configurator, atau pakai Smart Extractor (upload gambar untuk analisis AI).
